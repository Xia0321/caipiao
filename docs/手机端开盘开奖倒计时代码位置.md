# 手机端「开盘 / 开奖」倒计时代码位置

## 一、流程概览

- **首屏**：后端 `make.php` 算出 `pantime`（距封盘/开盘秒数）、`othertime`、`kjtime`（距开奖秒数），写入模板。
- **模板**：`makev.html` 把这三个值放到 `.close` 的 `time0`/`time1` 和 `.open` 的 `timek` 上。
- **前端**：`makevuser.js` 从 DOM 读 time0/time1/timek，用 `time0x()`、`time1x()`、`timekx()` 每秒减一并刷新显示；为 0 或定时会调 `getnowtime()` 拉取最新数据。
- **刷新接口**：`getnowtime()` 请求 `time.php`（带 `xtype=getopen`），返回新的一串倒计时与状态，前端再更新 time0/time1/timek 并重新跑倒计时。

## 二、后端（PHP）

### 1. 首屏：计算并传入模板

**文件**：`mxj/make.php`

| 行号 | 说明 |
|------|------|
| 87 | 查询当前期 `opentime, closetime, kjtime` |
| 89–97 | 根据 `panstatus` 算 **封盘/开盘倒计时** `$pantime`（秒） |
| 98–106 | 根据 `otherstatus` 算 **其他盘** `$othertime`（秒） |
| 107–110 | 若未自动开盘等，把 pantime/othertime 设为 9999 |
| 112 | **开奖倒计时** `$kjtime = strtotime(kjtime) - time()` |
| 113–117 | `assign('panstatus','otherstatus','kjtime','pantime','othertime')` 给模板 |

### 2. 刷新接口：返回最新倒计时（供前端 getnowtime）

**文件**：`mxj/time.php`

| 行号 | 说明 |
|------|------|
| 7–9 | 查当前期 `opentime, closetime, kjtime` |
| 11–16 | 若当前期未到开盘时间，回退到上一期 |
| 17–24 | 计算 **封盘/开盘** 剩余秒数 `$pantime` |
| 25–32 | 计算 **其他盘** `$othertime` |
| 33–36 | 不自动开盘时 pantime/othertime 置 9999 |
| 37 | **开奖倒计时** `$kjtime = strtotime(kjtime) - time()` |
| 56 | 输出：`pantime|othertime|thisqishu|panstatus|otherstatus|Hi|check|kjtime`（前端按 `m[0]`=pantime, `m[1]`=othertime, `m[7]`=kjtime 解析） |

### 3. 其他：makelib.php 的 getopen（非当前倒计时显示用）

**文件**：`mxj/makelib.php`

| 行号 | 说明 |
|------|------|
| 790–808 | `case "getopen"`：只查并返回 `panstatus`、`otherstatus` 的 JSON，**不返回 pantime/othertime/kjtime 秒数**。手机端倒计时刷新用的是 `time.php`，不是这里。 |

## 三、前端（模板 + JS）

### 1. 模板：把后端给的秒数写到 DOM

**文件**：`templates/default/mxj/makev.html`

| 行号 | 说明 |
|------|------|
| 3763 | **开盘/封盘**：`<div class="close ..." time0='{+$pantime+}' time1='{+$othertime+}' ...>`，内层 `<span class="mxj-timer-value">00:00</span>` 由 JS 每秒更新 |
| 3764 | **开奖**：`<div class="open ..." timek='{+$kjtime+}'>`，内层 `<span class="mxj-timer-value">00:00</span>` 由 JS 更新 |
| 1538–1545 | 样式：`.jvJTfN .close .mxj-timer-value`（红）、`.jvJTfN .open .mxj-timer-value`（灰） |

### 2. JS：读秒数、倒计时、请求刷新

**文件**：`js/default/jsmxj/makevuser.js`

| 行号 | 说明 |
|------|------|
| 1 | 全局变量：`time0, time1, timek, settime0, settime1, settimek` |
| 68–70 | **从 DOM 取初始秒数**：`time0 = .jvJTfN .close attr("time0")`，`time1 = attr("time1")`，`timek = .jvJTfN .open attr("timek")` |
| 71–77 | 取后若为负则取绝对值，然后调用 `time0x()`、`timekx()`，fenlei==100 时再调 `time1x()` |
| 2044–2119 | **getnowtime()**：请求 `mulu+'time.php'`，`data: 'xtype=getopen'`，成功回调里用 `m = m.split('|')`，`m[0]`=pantime、`m[1]`=othertime、`m[7]`=kjtime，更新期号/封盘·开盘文案，并 `time0 = m[0]; time1 = m[1]; timek = m[7];` 后调用 `time0x(); time1x(); timekx()` 重新倒计时 |
| 2133–2189 | **time0x()**：每秒 `time0--`，格式化为 `m:s`，写入 `.jvJTfN .close .mxj-timer-value`（**开盘/封盘倒计时**）；time0==0 时调 `getnowtime()` |
| 2206–2225 | **time1x()**：仅 fenlei==100 时用，每秒 `time1--`，写入同一 `.close .mxj-timer-value`（另一盘状态） |
| 2228–2249 | **timekx()**：每秒 `timek--`，格式化为 `m:s`，写入 `.jvJTfN .open .mxj-timer-value`（**开奖倒计时**）；若本期与上期非连续则显示 "00:00" |

## 四、数据对应关系

| 含义 | 后端变量 | 模板属性 | 前端变量 | 显示位置 |
|------|----------|----------|----------|----------|
| 封盘/开盘倒计时 | pantime | .close time0 | time0 | 封盘 0:42（红） |
| 其他盘倒计时（如 fenlei=100） | othertime | .close time1 | time1 | 同上 .close |
| 开奖倒计时 | kjtime | .open timek | timek | 开奖 0:58（灰） |
